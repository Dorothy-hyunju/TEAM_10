# 🛏️ 매트리스 구매 가이드 AI Agent

사용자의 수면 습관, 체형, 예산을 종합 분석하여 최적의 매트리스를 추천하는 지능형 AI Agent입니다.

## ✨ 주요 기능

### 🎯 **개인화된 추천**
- 수면자세, 체중, 건강 상태 분석
- 예산 범위 내 최적 옵션 제시
- 개인별 맞춤 점수 계산

### 🔍 **RAG 시스템**
- ChromaDB 기반 벡터 검색
- Sentence Transformers 임베딩
- 유사도 기반 매트리스 매칭

### 🤖 **Function Calling**
- 체계적인 5단계 추천 프로세스
- 고급 Prompt Engineering 적용
- 논리적 추천 근거 제시

### 🌐 **웹 인터페이스**
- Streamlit 기반 사용자 친화적 UI
- 실시간 채팅 상담
- 시각화된 추천 결과

## 🛠️ 기술 스택

### **AI/ML**
- **LLM**: Hugging Face Transformers
- **임베딩**: Sentence-Transformers
- **벡터DB**: ChromaDB
- **프롬프트**: Advanced Prompt Engineering

### **웹 프레임워크**
- **Frontend**: Streamlit
- **시각화**: Plotly
- **데이터 처리**: Pandas, NumPy

### **개발 환경**
- **언어**: Python 3.8+
- **IDE**: VS Code, Google Colab
- **버전 관리**: Git, GitHub
